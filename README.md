## ATARI POKEY AY/YM player

[SAPRTOOLS](https://github.com/ivop/saprtools)

convert YM to POKEY:

ym2sapr filename.ym

lzss -6 left.sapr left.lz16

lzss -6 right.sapr right.lz16

mads -o:filename.xex asm\playlzs16-stereo.asm
